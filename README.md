# 냥장고(Nyangjanggo)


### 여러분들의 소중한 음식 재료들이 냉장고 안에서 잠자고 있나요?

- 재료들을 내 손안의 냉장고에 넣어 두고 관리하세요!
- 냥장고는 사용자의 냉장고에 들어있는 재료들을 가지고 만들 수 있는 여러 사람들이 올린 다양한 레시피를 구경해보세요!
- 귀찮음 많은 여러분들을 위한 쉽고 빠르고 정확한 검색 서비스를 제공합니다.
